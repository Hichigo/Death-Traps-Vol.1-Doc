# Changelog

## **1.2.0 - 14-03-2021**

### Changed

* Reworked the damage system, now it is done using standard damage events (apply point damage)
* Added damage type DamageType for compatibility with [Death Traps Vol.2](https://hichigo.gitbook.io/death-traps-vol-2/)

## **1.1.0 - 20-10-2019**

### **New**

* Added blueprint interface **"BPI\_CustomDamage"** for unification of damage from trap to creatures and an easier way to damage new types of creatures or existing ones. This system is the same for [Death Traps Vol.1](https://www.unrealengine.com/marketplace/en-US/slug/death-traps-vol) and [Death Traps Vol.2](https://www.unrealengine.com/marketplace/en-US/slug/death-traps-vol-01).
