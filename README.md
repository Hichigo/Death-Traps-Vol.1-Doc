# Introduction

## Description

Death traps animated by curves. The package contains more than 10 ready-made traps. The package also contains basic Blueprints for creating your own traps, you need to create a grid, a curve for animation and adjust some parameters such as damage, number of repetitions, triggering, etc. (more details in the documentation) For faster tests when creating traps, a tuned character with health and a camera with a first / third person view is included in the package.
