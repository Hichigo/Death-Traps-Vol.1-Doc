# BP\_OpenFloor

## Settings

### **Animation**

**Rotation By Curve**

* **RotationStart\_Mesh\_1** (Rotator): start of rotation of the first mesh based on “_**CurveVectorAnimation**_”
* **RotationEnd\_Mesh\_1** (Rotator): end of rotation of the first mesh based on “_**CurveVectorAnimation**_”
* **RotationStart\_Mesh\_2** (Rotator): start of rotation of the second mesh based on “_**CurveVectorAnimation**_”
* **RotationEnd\_Mesh\_2** (Rotator): end of rotation of the second mesh based on “_**CurveVectorAnimation**_”
