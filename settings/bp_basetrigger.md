# BP\_BaseTrigger

A trigger, a link to which you can put traps to activate them. It creates an “**Trigger\_Disp**” (Event Dispatcher) to activate a trap subscribed in “**BP\_BaseTrap**”.

## Settings

* **UseShapeCollidsion** (E\_ShapesCollision): based on this variable creates a collision form to choose from 3:
  * Box
  * Sphere
  * Capsule
* **TransformShapeCollision** (Transform): Position, rotation and size of the collision
