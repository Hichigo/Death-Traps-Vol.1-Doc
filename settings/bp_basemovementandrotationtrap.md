# BP\_BaseMovementAndRotationTrap

## Settings

* **WeaponClass** (BP\_BaseModularWeaponClass): trap “weapon” class

### **Animation**

**Rotation By Curve**

* **RotationStart** (Rotator): start of rotation based on “_**CurveVectorAnimation**_”
* **RotationEnd** (Rotator): end of rotation based on “_**CurveVectorAnimation**_”

**Location By Curve**

* **LocationStart** (Vector3D): start moving based on “_**CurveVectorAnimation**_”
* **LocationEnd** (Vector3D): end of moving based on “_**CurveVectorAnimation**_”
